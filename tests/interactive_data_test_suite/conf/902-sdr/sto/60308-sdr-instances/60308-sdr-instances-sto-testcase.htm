603-08 : EDGAR Filer Manual v62 6.3.8 page 6-7

Created by: SEC Office of Structured Disclosure

A submission must contain exactly one EX-101.INS or EX-99.SDR K.INS.

An Interactive Data instance in XBRL format must be an EX-101.INS, EX-99.SDR K.INS, or EX-99.SDR L.INS.

REMARK: This is a continuation of 603-08.
