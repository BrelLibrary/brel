610-03 : EDGAR Filer Manual v68d 6.10.3

Created by: SEC Office of Structured Disclosure

US English labels are required.

If an element used in an instance is assigned a label in the DTS whose xml:lang attribute does not equal 'en-US', then the DTS must also contain a link:label for the same element and all other attributes with an xml:lang attribute equal to 'en-US'. Non-US English labels may appear for elements in the schema, but they must be translated into US English if they are used in an instance.
